# Automated-Time-Table-Generator
**Project Overview**

The Automated Timetable Management System is a web-based application designed to automate the creation, management, and distribution of timetables for educational institutions and small city services such as hospitals and transportation systems. The system eliminates manual scheduling, reduces conflicts, and ensures efficient utilization of available resources.

This project provides a centralized platform where administrators can generate timetables based on predefined constraints, while users can view schedules in a simple and user-friendly interface.


**Role-Based Authentication
**
Admin login for add/update/delete operations

Public access for viewing timetables

Real-Time Updates to schedules

Downloadable Timetables in PDF / Excel formats

MVC Architecture for clean code structure and scalability

**Technology Stack
**
Backend: Java (Maven), MVC Architecture

Frontend: HTML, CSS, JSP

Database: MySQL

IDE: Eclipse
